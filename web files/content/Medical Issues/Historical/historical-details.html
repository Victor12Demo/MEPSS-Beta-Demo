Historical Overview

The historical section of the secondary menu lists prior rating schedule criteria for several diagnostic codes; including those diagnostic codes that no longer exist. You will be given:

  • The old diagnostic code
  • The original name
  • The new diagnostic code that replaced it (unless it says it has been deleted)
  • The date of the old criteria
  • The old rating schedule criteria

This will prove useful when working with cases that contain older, outdated diagnostic codes.
